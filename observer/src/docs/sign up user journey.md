# User Journey: Create a New JustServe Account

## Target Audience
- Primary persona: New JustServe end user (volunteer)
  - All Languages and Devices

## User’s Context & End Goal
#### Context: 
User doesn't have a JustServe account and is visiting JustServe for the first time
#### End Goal 
- User has an account and is signed in to said account on JustServe
- Account record is created once, credentials are stored correctly, and verification status is accurate.

## Journey Stages
1. Entry
2. Account Creation
3. Verification & Access

## Actions (Action → Result)
1. 1. User selects an option to go to a login form from the home page. 
2. 1. User enters required information and selects **“Create Account”** → System validates inputs and submits account request.
3. 1. User Proceeds to be logged in and views main page content relative to their profile location <br> <blockquote> [Needs verification - is 3.1 based on their IP or profile location?] </blockquote
